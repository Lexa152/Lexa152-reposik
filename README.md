# Проект виджета банковских операций

Предназначен для учебных делей при изучении языка программирования "Python"

## "Особенности" проектика:
1. по ходу работы используются стандартные библиотеки;
2. проведено тестирование всего функционала модулем PyTest;
3. для работы требуется только скачать: https://github.com/Lexa152/Lexa152-reposik.git;
4. реализованы функции для работы с массивами транзакций.

## Минимально необходимые зависимости проекта:
1. requests (>=2.34.2,<3.0.0);
2. Интерпретатор Python версии 3.10

## Выполняемые функции:
Пока только частично скрытое отображение номеров банковских счетов и карт.
